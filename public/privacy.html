Coderoot Privacy Policy

Last updated: May 15, 2026

Coderoot is a Chrome Extension that adds advanced concept notes to supported Codetree introduction pages. This policy explains what information Coderoot handles and why.

Information Coderoot handles

  • Codetree page context, such as the current page URL, site language, problem slug, and selected problem language, to find the matching XML note.
  • XML note content that an authorized contributor writes or edits in the Coderoot editor.
  • GitHub account identity returned through GitHub OAuth, such as GitHub username, to verify whether a contributor is allowed to save XML content.
  • A short-lived Coderoot session token stored by the extension after GitHub verification. Coderoot does not store GitHub personal access tokens in the extension.

How the information is used

Coderoot uses this information only to provide its single purpose: loading, previewing, editing, and saving Coderoot XML concept notes for Codetree pages.

When a contributor saves XML content, the deployed Coderoot API writes the change to the Coderoot content repository through the installed GitHub App. Approved XML content may become public in the Coderoot content repository.

Remote content and code

Coderoot fetches XML notes and GitHub metadata from remote services as data. The extension does not execute remotely hosted JavaScript. Extension JavaScript and CSS are packaged inside the extension.

Sharing and sale of data

Coderoot does not sell user data. Coderoot does not transfer user data to third parties for advertising, creditworthiness, or unrelated purposes.

Data retention

Coderoot session data may be stored locally by the extension and can be cleared by removing the extension or clearing extension storage. XML content saved to GitHub follows the retention and history behavior of the Coderoot content repository.

Contact

For questions, open an issue at https://github.com/kommiter/coderoot.
